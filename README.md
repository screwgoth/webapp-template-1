# Web Application Template

A production-ready, full-stack web application template with authentication, user management, and Docker deployment.

## Tech Stack

- **Frontend**: React 18 + TypeScript + Vite + Tailwind CSS
- **Backend**: Node.js 20 + Express + TypeScript
- **Database**: PostgreSQL 16 + Prisma ORM
- **Deployment**: Docker + Docker Compose

## Quick Start

```bash
# Clone the template
git clone https://github.com/screwgoth/webapp-template-1.git my-new-project
cd my-new-project

# Setup environment
cp .env.example .env

# Start with Docker
docker compose up -d

# Or start development
npm install
npm run dev
```
